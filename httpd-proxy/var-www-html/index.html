Shibboleth IdP - OpenID Connect Home

This is an instance of the Shibboleth Identity Provider with support for OpenID Connect, running inside Docker. The running container presents the identity provider as well as numerous sample applications that are pre-configured to interact with the identity provider via SAML or OpenID Connect.
Development is sponsored by the University of Chicago.
